⌂ How to use Tarkov Stash Helper

1 — First-time setup
  1. Launch Tarkov and open your stash.
  2. Click Set Stash Region in the header — a dark overlay will appear over this browser window.
  3. Click and drag a rectangle over the area of your screen that contains the stash grid, then release. The coordinates are saved automatically.
  4. You only need to do this once unless you change your monitor layout.
2 — Scanning your stash
  1. Open your stash in Tarkov (keep the game focused so lossless scaling is active).
  2. Press … — the screenshot fires instantly while the game is still in the foreground. You can change the hotkey in ⚙ Settings.
  3. Tab to this window — the scan result is already loaded.
  4. On this page: green highlight = keep-list item you've already acquired, red highlight = keep-list item still needed, amber highlight (not FIR) = a Kappa item found but not Found-in-Raid, so it can't be handed in yet. On the Sell Advisor: cyan = KEEP (needed for kappa/tasks — don't sell), green = flea, gold = trader.
  5. If a scan finds keep-list items you haven't checked off, a cyan bar appears — click Mark acquired to check them all at once.
  6. Alternatively, click Scan Stash manually (will capture at 720p if game isn't focused).
3 — Keep list
  • Check the box next to an item once you've found it or turned it in — it'll grey out and stop appearing as a keep target in scans.
  • Use the search bar to filter by item name or task.
  • Add custom items with the form at the bottom — optionally set aliases (comma separated) so OCR can recognise the in-game abbreviated name.
  • Items with a FIR badge must be Found In Raid when handed in.
  • Click × on any row to permanently remove an item.
4 — Prestige
  • Set your current prestige level in the header dropdown.
  • At prestige 1, New Beginning figurines do not need to be FIR — the badge is hidden.
  • At prestige 2 and above, all New Beginning figurines must be FIR — the amber FIR badge appears.
5 — OCR tips
  • OCR works best when the stash region is set tightly — don't include the inventory/gear panels.
  • If an item isn't detected, add its in-game short name as an alias via the add form (e.g. "FireKlea" for FireKlean gun lube).
  • Requires Tesseract OCR to be installed: winget install UB-Mannheim.TesseractOCR
  • Closing this window just minimizes it to the system tray — right-click the tray icon to reopen or quit.
